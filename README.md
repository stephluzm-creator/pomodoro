# Pomodoro Timer

A clean, minimal Pomodoro timer app built with Python and Tkinter.

## Features

- 25-minute work sessions with 5-minute short breaks and 15-minute long breaks
- Visual countdown ring with live timer display
- Tracks completed pomodoros with a dot indicator
- Sound notification when a session ends
- Start, pause, reset, and skip controls

## How it works

The Pomodoro Technique breaks work into focused 25-minute sessions separated by short breaks. Every 4 sessions, you get a longer 15-minute break.

## Requirements

- Python 3
- Tkinter (included with Python on Mac)

## Run

```bash
python3 pomodoro.py
```
